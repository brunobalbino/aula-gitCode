# aula-gitCode 

* Criei o novo repositório **aula-gitCode** no github



* Criei a pasta localmente com o mesmo nome **aula-gitCode** 



* Criei o arquivo **README.md**



* No Git Bash iniciei o projeto através do comando:



`git init`



* Adicionei o arquivo para ser feito o commit através do comando:



`git add README.md`



* Realizei o commit através do comando:
 


`git commit -m "Criado arquivo README.md"`



* Vinculei o diretório local com o diretório remoto
 


`git remote add origin https://github.com/brunobalbino/aula-gitCode.git`

* Postei o conteúdo do diretório local para o servidor do github 



`git push -u origin master`
